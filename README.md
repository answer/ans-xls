# Ans::Xls

xls mime type を追加、 xml spreadsheet を出力する layout と helper を提供

## Installation

Add this line to your application's Gemfile:

    gem 'ans-xls'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ans-xls
